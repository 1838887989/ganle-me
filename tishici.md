设计一个待办任务管理应用的 UI 界面。

风格要求：
- 有机生物亲和设计风格，自然温暖的大地色调
- 主色调：森林绿 (#228B22) 和橄榄绿 (#6B7B3C)
- 背景色：柔和奶油色 (#F5F0E1)
- 强调色：收获金 (#CA8A04)
- 圆角：16-24px，有机曲线感
- 阴影：柔和自然阴影，无硬边
- 字体：Lora (标题) + Raleway (正文)

功能模块：
- 顶部导航栏：应用名称、主题切换按钮（明/暗模式）、窗口控制
- 任务输入区：输入框 + 时间选择 + 添加按钮
- 任务列表：复选框 + 任务名称 + 编辑/删除操作
- 侧边栏：项目分类、智能清单、标签筛选

设计原则：
- 简洁清晰，减少视觉噪音
- 使用 SVG 图标（Lucide Icons），不使用 emoji
- 所有可点击元素添加 cursor-pointer
- 悬停状态使用颜色/透明度变化，避免缩放
- 支持明/暗模式切换，确保两种模式下对比度符合 WCAG AA 标准
